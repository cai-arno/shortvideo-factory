# 生产部署指南

> 短视频工厂系统 - 一键部署手册

## 服务器要求

| 配置 | 最低 | 推荐 |
|------|------|------|
| CPU | 2 核 | 4 核 |
| 内存 | 4 GB | 8 GB |
| 磁盘 | 50 GB | 100 GB SSD |
| 系统 | Ubuntu 22.04 LTS | Ubuntu 22.04 LTS |

## 部署步骤

### 1. 服务器初始化

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装 Docker & Docker Compose
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 安装 Git
sudo apt install -y git
```

### 2. 拉取代码

```bash
git clone https://github.com/your-org/shortvideo-factory.git
cd shortvideo-factory/infrastructure/docker
```

### 3. 配置环境变量

```bash
cp .env.prod.example .env.prod
nano .env.prod  # 填入实际配置
```

### 4. 构建并启动

```bash
# 构建前端（生产镜像需要先推送）
docker-compose -f docker-compose.prod.yml build

# 启动所有服务
docker-compose -f docker-compose.prod.yml up -d

# 查看状态
docker-compose -f docker-compose.prod.yml ps
```

### 5. 数据库迁移

```bash
# 进入后端容器执行迁移
docker exec -it shortvideo-backend bash
alembic upgrade head
exit
```

### 6. SSL 证书（可选）

```bash
# 安装 Certbot
sudo apt install -y certbot python3-certbot-nginx

# 申请证书
sudo certbot --nginx -d your-domain.com
```

## 维护命令

```bash
# 查看日志
docker-compose -f docker-compose.prod.yml logs -f

# 重启服务
docker-compose -f docker-compose.prod.yml restart

# 更新部署
git pull origin main
docker-compose -f docker-compose.prod.yml build
docker-compose -f docker-compose.prod.yml up -d

# 备份数据库
docker exec shortvideo-postgres pg_dump -U postgres shortvideo > backup_$(date +%Y%m%d).sql
```

## 防火墙配置

```bash
sudo ufw allow 22/tcp    # SSH
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
sudo ufw enable
```
